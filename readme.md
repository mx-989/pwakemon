# PWAkemon

Une application web pour capturer des pokémons.

### Usage

```bash
npm install
```

```bash
npm run build
```

```bash
npm run preview
```

Le server live sera accessible à l'adresse: `http://localhost:4173` (ou redirigié vers un autre port si celui-ci est déjà utilisé).
## Live Demo

The web app is accessible at: [https://mx-989.github.io/pwakemon/](https://mx-989.github.io/pwakemon/)
